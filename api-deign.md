# API Design

(游댮, 游리, 游릭)

|     | Path                     | Info                                       |
| :-: | ------------------------ | ------------------------------------------ |
| 游릭  | `GET /status`            | Gets the system status (System info, Logs) |
| 游리  | `GET /app/{app}`         | Gets info on an app                        |
| 游댮  | `POST /app/{app}/action` | Preforms an actions related to the app     |
